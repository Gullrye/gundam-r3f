# 3D Gundam RX-78-2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an interactive 3D RX-78-2 Gundam robot using React Three Fiber with WASD keyboard movement, walking animation, pose switching, auto animation, and color customization.

**Architecture:** Each body part is a standalone R3F component built from basic Three.js geometries. A hierarchical group structure connects parts at joints. A shared `jointRef` is updated in `Gundam.tsx`'s `useFrame` and read by each body part's own `useFrame` to directly mutate Three.js group rotations (bypassing React re-render). Keyboard input drives real-time movement and walking animation.

**Tech Stack:** Vite + React + TypeScript + @react-three/fiber + @react-three/drei

### Task 13: Walking Animation + Keyboard Controls [x] DONE

**Key architecture decision:** Body parts read from a shared `jointRef` in their own `useFrame` to directly mutate Three.js `group.rotation`, bypassing React re-render. This is necessary because `useFrame` mutations on refs don't trigger React re-renders.

**Keyboard controls:**
- **W** — walk forward (legs swing, arms swing opposite, body bobs)
- **S** — walk backward
- **A** — strafe left
- **D** — strafe right
- **Q** — turn left
- **E** — turn right

**Walking animation:**
- Sinusoidal joint oscillation via `getWalkAngles(time)`
- Legs: hip swings ±0.5 rad, knee bends on backswing up to 0.7 rad
- Arms: opposite swing to legs ±0.5 rad, elbows bend on forward swing
- Body: subtle vertical bob at 6Hz
